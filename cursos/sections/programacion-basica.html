El DOM y canvas


-Javascript puede manipular objetos de un navegador, principalmente el navigator, el window y el document. Esto se hace con el document object model (DOM)

-El dom es la forma interna en que el navegador organiza el html en una estructura especial, con javascript se puede maniuplar esto.

-El dom es el arbol de estructura del html


Dibujar en el DOM con canvas

-La etiqueta canvas permite dibujar en html, crea una caja con un ancho y con un altura que se especifica.

-Canvas funciona como un sistema de cordenadas normal, con la excepcion de que el origen va a ser en la esquina superior izquierda y todos los 'Y' verticales son puntos positivos

var draw = document.getElementById('id_canvas');
var lienzo = draw.getContext('2d'); {getContext es una funcion de la etiqueta canvas, asi se le especifica si es un dibujo en 3d o 2d}

lienzo.beginPath(); {funcion para iniciar el dibujo}

lienzo.strokeStyle = 'red'; {Definir el color }

lienzo.lineWidth = 3; {Define el ancho de la linea en px}

lienzo.moveTo(100, 100); {Definir el origen del trazo}

lienzo.lineTo(200, 200); {definir la cordenada final de una linea}

lienzo.stroke(); {levantar el lapiz}

lienzo.closePath(); {Terminar el dibujo}


Eventos y formularios


Un evento es un suceso que ocurre en determinado momento: click, hover, focus, pulsar una teclam etc...

-La funcion se llama addEventListener y recibe 2 parametros, el tipo de evento y la funcion a invocar cuando sucede el evento.

-Cuando la funcion del evento recibe un parametro, este tendra toda la nformacion del evento como un objeto, por lo tanto se le puede hacer un console.log y mirar cuales son sus propiedades o atributos.

-Para que una funcion se invoque con un evento, se nesecita hacer esto:

var variable = document.getElementById('nombre');
variable.addEventListener('click', funcion);

-Para obtener algun valor o caracteristica de un formulario de html, se puede hacer un console.log al objeto y asi poder ver cuales son sus metodos y atributos, esto es aplicable pra cualquier etiqueta de html.


Tipos de eventos:

"click" {Al hacer click}

"keydown" {Al presionar cualquier tecla}

"keyup" {Al soltar cualquier tecla}

"mousedown" {Cuandoo se pulsa el mouse, click}

"mousemove" {Cuando se mueve el mouse}

"mouseup" {Cuando se levanta el click del mouse}

"load" {Cuando carga algo, como una imagen}

Usar un servidor con express en pc


-Node js es una manera de correr Javascript del lado del servidor. { Se requieren tener permisos de administrador para ejecutar el instalador y utilizar el Node}

-Versiones de Node JS:

LTS => Version mas estable, utilizada por la mayoría de los usuarios corporativos
Current => Contiene los últimos features.

- IPCONFIG(win) = IFCONFIG(Mac/Linux) => Visualizar la configuración de Red del computador, para ver la ip

NPM => Sistema que ayuda a Node a instalar Librerías.

Instalar Servidor Express en el entorno de Node de manera global => npm install express -g

Require => Es una función que busca dentro del equipo las dependencias pertenecientes a Express y las almacena en la instancia creada para el efecto

Request => Petición realizada al servidor.

Response => Respuesta del Servidor al Request enviado.

Listen (8080) => Pone el servidor en modo escucha, en el puerto 8080 en este ejemplo, el puerto es personalizable.

En Windows especialmente el firewal puede presentar restricciones en cuanto a acceso a puertos, por lo que en ocasiones puede haber navegación al servidor de manera local, mas no asi de manera remota, en ese caso realizar la siguiente verificación de permisos: INICIO/FIREWALL DE WINOWS/PERMITIR A UNA APLICACIÓN O CARACTERÍSTICA SU EJECUCIÓN/NODE JS SERVER SIDEJAVASCRIPT.


Circuitos electronicos


La electricidad es un flujo de electrones medido en voltajes y en amperimetro. La electricidad es creada en plantas de energía y se almacena en las baterias (carro, celular, robots, etc) Se transmite por cables hasta la energía de tu casa.

Los circuitos electrónicos son la base de la tecnología moderna

Electricidad => Son el flujo constante de electrones a través de un cable. Se mide en voltaje y amperaje

Voltio => Intensidad con que la corriente se mueve por el cable.

Resistencia => Controla la intensidad de la corriente eléctrica, son medidas en Ohmios.

Circuito digital => Convierte la electricidad en 0 y 1.

Memoria RAM => Almacena los 0 y 1 mientras tenga energía

Discos Duros => Almacena los 0 y 1 aun sin energía
