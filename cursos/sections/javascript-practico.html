.reduce() .sort()


.reduce():

-reduce es un metodo de arrays para poder agrupar los datos de este, sirve para sumarlos, multiplicarlos, etc...

const sum = list.reduce(function(acomulado = 0, elemento){

return acomulado + elemento;

});

-La funcion anonima de reduce nesecita 2 parametros, el primero es el valor acomulado de cada iteracion y el segundo es el que recorre cada elemento

-El valor acomulado tiene que inicialisarse en 0 para que en la primera iteracion no ocurra ningun error


.sort():

list = list.sort(function (a, b) {

return a - b;
});

-El metodo sort hace que se ordene los elementos de forma ascendente, funciona igual que reduce con los 2 parametros pero este tiene una reglas de validacion interna que hacen que se ordene siguendo esta estructura en el return.
