Que es javascript


-Javascript es un lenguaje interpretado, orientado a objetos, débilmente tipado y dinámico. Nace con la necesidad de generar dinamismo en las páginas web y que a su vez los usuarios y las empresas pudieran interactuar unos con otros.

Debilmente tipado = se pueden hacer estas cosas:

4 + "7"; // 47
4 * "7"; // 28
2 + true; // 3
false - 3; // -3


Dinamico:

-Corre directamente en la etapa de Runetime sin una etapa de compilación previa(compilar = traducir el lenguaje de prog a binario). Esto permite probar nuestro código inmediatamente; pero también es lo que hace que los errores se muestren hasta que se ejecuta el programa.


Lenguaje interpretado:

-El navegador lee linea por linea nuestro código el cuál le indica lo que tiene que hacer, sin la necesidad de compilar. Todo esto es controlado por el motor de Javascript V8 del navegador


Backwards compatible:

-Todas las funciones nuevas que salen de Javascript no dañarán el trabajo ya hecho, pero no se podrá utilizar en nuestro entorno de trabajo inmediatamente. Para solucionar esto está Babel que permite utilizar las nuevas características del lenguaje pero lo transforma a una versión que el navegador pueda entender.

ParseInt vs Number


parseInt:
-Analiza la cadena desde el primer dígito, hasta que encuentre algo que no sea dígito y devuelve lo que haya analizado.

Ejemplo: parseInt(“123hui”); = 123
Ejemplo: parseInt(“hui123”); = NaN
Ejemplo: parseInt(“1hui23”); = 1

Number:
-Number: Busca convertir toda la cadena en un número, por lo que al encontrarse con un elemento que no sea digito nos dara como resultado NAN (Not A Number).

Ejemplo: Number(“123hui”); = NaN
Ejemplo: Number(“123”); = 123

Filter y map

Filter: Permite filtrar los elementos de un array y devolverlos en un nuevo array.
Map: Permite seleccionar atributos o propiedades para ponerlos en un nuevo array.

-Para ambos se nesecita generar un nuevo array dado que estos metodos no modifican el array original.

Filter:

var articulos = [
{nombre: 'Balon', costo: 50000},
{nombre: 'Bici', costo: 300000},
{nombre: 'Libro', costo: 80000},
{nombre: 'Camisa', costo: 80000},
{nombre: 'Zapato', costo: 400000},
];

var filtrados = articulos.filter(function(elemento){
return elemento.costo <= 100000;
});

console.log(filtrados);


Map:

var mapeados = articulos.map(function(elemento){
return elemento.nombre
});

console.log(mapeados);


.find() .forEach() y .some()


.find():

var findeados = articulos.find(function(elemento){
return elemento.nombre === "Bici";
});

-find retorna el primer elemento que encuentre de un array segun la condicion, filter retorna todos los elementos que encuentre, en vez de solo el primero.


.forEach():

articulos.forEach(function(elemento){
console.log(elemento.nombre);
});

-Es lo mismo que map pero en este no es necesario crear un nuevo array


.some():

var someados = articulos.some(function(elemento){
return elemento.costo <= 50000;
});

-Solo devuelve un valor booleano, en caso de que la condicion exista en alguna parte del array, la variable sera = true
