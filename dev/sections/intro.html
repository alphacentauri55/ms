Como y porque aprender Data science & AI


Data science:

-Data science es el proceso de descubrir información valiosa de los datos.

-Tomar decisiones y crear estrategias de negocio

-Crear productos de software más inteligentes y funcionales.

-Obtencion de datos, medicion, encuestas, internet, Transformar y limpiar datos, explorar y analizar datos, visualizar datos, patrones, tendencias, graficos, reportes

-Integrar datos e IA a productos de software, Ponerlos a disposición del usuario final.

Proceso de la ciencia de datos: Método científico llevado a los datos, El proceso entre proyecto a proyecto cambia poco.

"Data science es una intersección de los conocimientos matemáticos, programación e interpretación de los datos"


Inteligencia artificial: Algoritmos para predecir eventos futuros que emulan cognicion humana

Big data: Grandes volumenes de datos muy variados y muy veloces que no es facil procesarlos de manera tradicional. Para un conjunto de datos ser llamado big data tiene que tener las 5 V (Volumen, velocidad, variedad, valor, veracidad).

-Se suelen usar servicios de computo en la nube.

-Big data es la materia prima que podemos usar en data science para hacer un analizis mas exhaustivo


Que NO es data science: No es magia - No es Inteligencia Artificial - No es tener solo métricas de un dato sin tener alguna métrica de valor - No son puras matemáticas - No es solo trabajar con Big Data.

Jerarquia de nesecidades de data science:

1. Etapa de recolección de los datos: Los datos pueden venir de diversas fuentes dependiendo de la industria (páginas web, sensores, instrumentación…)

2. Etapa de Almacenamiento/Movimiento: Los datos se deben movilizar y almacenar a través de una infraestructura o un pipeline.

3. Etapa de Exploración y transformación: Se deben limpiar los datos, detectar errores y anomalías y prepararlos para un posterior análisis.

4. Etapa de Análisis: Se buscan métricas, segmentaciones, agregaciones y distintas características para convertir los datos en información y posteriormente en conocimiento.

5. Etapa de IA: Se generan predicciones y se visualiza el futuro de la organización.


Que hace un data analist: Utiliza los datos para obtener Insights. es decir info de valor que permita y ayude en la toma de decisiones y planteamiento de estrategias de negocio.
Esto lo realiza con la extracción de datos recolectados, análisis de datos y reporte de resultados con con un formato amigable → dashboard, tableros, métricas.
Herramientas de data analist: Aqui

-Ingles
-Matemática (Algebra Lineal, Estadística etc)
-Programación (Python , R, etc)
-Base de Datos
-Conocimiento del Negocio
-Storytelling y Visualización de Datos
-Comunicación en General.


Data engineer - Data analist - Data scientist - Machine learning engineer

Data engineer: Preparacion y busqueda de datos mediante modelos ETL

Data analist: Aalisis y recopilacion de puntos clave sobre los datos

Data scientist: Proceso de análisis de datos para encontrar información de valor, que ayuden a tomar decisiones al negocio o crear productos basados en datos.

Machine laerning engineer: Automatisacion y creacion de modelos de machine learning en base a los datos y analisis del data science
